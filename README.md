# DevArt Article Photo for Joomla

Lightweight Joomla 6 component for generating optimized article-ready images with crop, zoom, logo and watermark support. Designed for high-performance news and editorial websites.

## Features

- Generate article-ready JPG images from JPG, PNG and WEBP uploads
- Output size selection per image: 1200 × 800 or 1620 × 1080
- Crop position and zoom controls
- Auto Preview without full page refresh
- Optional logo overlay
- Optional center watermark overlay
- Download generated image
- Save to server under `/images/YYYY/MM/`
- Latin-only filename prefix validation
- Joomla Options for JPG quality, save folder, default prefix and max source size
- Joomla ACL permissions

## Requirements

- Joomla 6+
- PHP 8.2+
- PHP GD extension with JPEG/PNG/WEBP support
