# DevArt Article Photo for Joomla

![Joomla](https://img.shields.io/badge/Joomla-6.x-blue)
![PHP](https://img.shields.io/badge/PHP-8.2%2B-green)
![License](https://img.shields.io/badge/License-GPLv3-red)
![Release](https://img.shields.io/badge/Version-1.0.0-orange)

Lightweight Joomla 6 component for generating optimized article-ready images for news and editorial websites.

DevArt Article Photo helps editors quickly prepare clean, lightweight and properly sized article images directly inside Joomla.

---

## Features

✅ Generate article-ready JPG images from JPG, PNG and WEBP uploads  
✅ Per-image output size selection:
- 1200 × 800
- 1620 × 1080

✅ Crop position and zoom controls  
✅ Auto Preview without full page refresh  
✅ Optional logo overlay  
✅ Optional center watermark overlay  
✅ Download generated image instantly  
✅ Save directly to server under `/images/YYYY/MM/`  
✅ Optimized JPG compression for smaller file sizes  
✅ Latin-only filename prefix validation  
✅ Responsive administrator interface  
✅ Joomla ACL permissions support  
✅ Temporary preview cleanup system  
✅ Joomla Update Server support

## Requirements

- Joomla 6+
- PHP 8.2+
- PHP GD extension with JPEG / PNG / WEBP support

---

## Installation

1. Download the latest release ZIP package
2. Go to Joomla Administrator → System → Install Extensions
3. Upload the package
4. Open **Components → DevArt Article Photo**

---

## Default Workflow

1. Upload image
2. Adjust crop / zoom
3. Enable logo or watermark if needed
4. Select output size
5. Generate Preview
6. Save to Server or Download

---

## Notes

- Default JPG compression is optimized for lightweight article images
- Generated files are organized automatically by year/month
- Recommended workflow for high-traffic Joomla news websites
- Always test on a staging environment before production use
